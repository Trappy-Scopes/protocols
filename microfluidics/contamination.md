# Contamination characterisation using fluorescence microscopy

## Aim

Identify contaminents in new microfluidics devices using multi-channel fluorescence microscopy. The rationale is that organic, metal, and PDMS dust particles have different spectral signatures.

## Procedure

1. New and clean microfluidics devices (INESC first batch of 50) were imaged without any fluid at 10X magnification using an epifluorescence confocal spinning disk microscope (https://www.intelligent-imaging.com/marianas).

	| Parameter     | Value                                                        |
	| ------------- | ------------------------------------------------------------ |
	| Microscope    | 3i Marianas (SlideBook acquisition)                          |
	| Objective     | 10× (NA not recorded in file)                                |
	| Pixel size    | 1.1 µm/px                                                    |
	| Field of view | 1200 × 1200 px (1320 × 1320 µm)                              |
	| Bit depth     | 16-bit (uint16)                                              |
	| Z / time      | Single plane, single time point, focused above the glass slide plane |

2. The  trap area was imaged for 5 devices. A brightfield acquisition was followed by excitation by a **405nm** laser and collection in four different filter channels.

	| Channel | Optical config | Modality                        | Excitation (nm) | Emission (nm)¹ | Exposure (ms) |
	| ------- | -------------- | ------------------------------- | --------------- | -------------- | ------------- |
	| 1       | c405s          | Autofluorescence                | 405             | ≈ 440–480      | 200           |
	| 2       | c488s          | Autofluorescence                | 488             | ≈ 510–545      | 200           |
	| 3       | cBF            | Brightfield (transmitted light) | —               | —              | 200           |
	| 4       | c561s          | Autofluorescence                | 561             | ≈ 590–620      | 200           |
	| 5       | c637s          | Autofluorescence                | 637             | ≈ 660–700      | 300           |

3. A ImageJ/FiJi macro was used to analyse the speaks. The code was generated by Claude and verified manually before use.

## Results

Most of the dust that was observed is organic in nature or PDMS. This is confirmed by autofluorescence. Some dust speaks show no autofluorescence (~10%) and are non-organic in nature. Adhesive from scotch tape shows up prominantly and is quite auto-fluorescent.
